Contextual Cache
================

You can configure the ``Twig Recorder`` with the following configuration:

.. code-block:: yaml

    twig:
        base_template_class: Sonata\CacheBundle\Twig\TwigTemplate13
        base_template_class: Sonata\CacheBundle\Twig\TwigTemplate14
